# Release History

## Purpose

Provides a high-level view of platform releases and major changes.

---

## Format

## vX.Y.Z

### Date
YYYY-MM-DD

### Included Versions

| Component | Version |
|----------|--------|

### Summary

- short description of what changed

### Breaking Changes

- list if any

### Notes

- migration notes
- operational changes

---

## Example

## v0.4.0

### Date
2026-04-10

### Included Versions

| Component | Version |
|----------|--------|
| API | v0.4.0 |
| Auth Server | v0.3.1 |
| Dashboard | v0.2.5 |
| Economy | v0.4.0 |
| Market | v0.3.0 |
| Deployment | v0.4.0 |

### Summary

- improved transfer idempotency
- standardized error semantics

### Breaking Changes

- API error format updated

### Notes

- dashboard must be updated to handle new error format
