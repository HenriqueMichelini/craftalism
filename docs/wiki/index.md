# Craftalism Wiki Index

## Purpose

This wiki is a navigation and context-compression layer for Craftalism docs.

It is **not** source of truth.

Canonical sources:
- [Governance Precedence](../governance-precedence.md)
- [System Summary](../system-summary.md)
- [Contracts](../contracts/)
- [Standards](../standards/)

## Start Here

- Read [docs/governance-precedence.md](../governance-precedence.md) first for authority and conflict resolution.
- Read [docs/system-summary.md](../system-summary.md) for system shape, repo roles, and ownership.
- Use the wiki pages below only to find the smallest relevant canonical read set.

## Wiki Pages

- [system-map.md](system-map.md): short map of repositories, ownership boundaries, and when to leave the root docs layer.
- [contracts-map.md](contracts-map.md): which shared contract to read, who owns it, and who consumes it.
- [standards-map.md](standards-map.md): which standard applies to a task and where its canonical file lives.
- [task-routing.md](task-routing.md): minimal reading paths for common task types.
- [status-map.md](status-map.md): current status pointers and historical audit routing.

## Rules

- Do not treat wiki pages as authority over contracts, standards, or repo-local docs.
- Do not use the wiki to infer undocumented behavior.
- Use audit files only when explicitly routed for historical evidence or verification context.
