\section{Introduction}

\subsection{Purpose of This Document}

This document describes the architecture, components, and development practices of the Craftalism ecosystem.

\subsection{Target Audience}

\begin{itemize}
	
	\item Developers
		\begin{itemize}
			\item People writing code (plugin, API, dashboard).
			\item They care about architecture, APIs, data flow, and setup.
		\end{itemize}
		
	\item Maintainers
		\begin{itemize}
			\item People who review PRs, cut releases, fix bugs.
			\item They care about conventions, testing, and long-term decisions.
		\end{itemize}
		
	\item Infrastructure
		\begin{itemize}
			\item People deploying and running the system.
			\item They care about Docker, environments, CI/CD, and monitoring.
		\end{itemize}
	
\end{itemize}
